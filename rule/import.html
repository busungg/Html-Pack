@import 규칙은 CSS파일에서 다른 CSS 파일을 추가하는 방법입니다.

CSS 파일의 규모가 커지면 코드를 쉽게 알아보기 위해 파일을 분리합니다. 파일을 분리하면 분리할수록 HTML 페이지 내부에 link 태그가 많아지므로 전체적인 코드가 지저분해집니다.
이때 link 태그의 숫자를 줄일 수 있는 방법이 바로 @import 규칙 입니다.
